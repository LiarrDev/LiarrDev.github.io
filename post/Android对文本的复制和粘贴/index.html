Android 对文本的复制和粘贴

很多电商类的 App 现在都会用到一键复制或者一键粘贴的功能，比如电商类的 App，支持一键复制物流单号，省去用户选择再复制的操作，十分方便。

一键复制和一键粘贴的操作如何实现呢？Android 为我们提供了剪贴板框架，允许复制和粘贴不同类型的数据，比如文本、图像、二进制流数据或者其他复杂的数据类型，另外，它还支持复制多条记录，很多输入法都实现了多条记录的剪贴板，功能很强大。

当然，最常用的还是对文本进行操作啦，那就介绍一下复制和粘贴文本的实现方法：

@Override
public void onClick(View v) {
    ClipboardManager clipboardManager = (ClipboardManager) getSystemService(CLIPBOARD_SERVICE);
    switch (v.getId()) {
        case R.id.copy:
            String s = textView.getText().toString();
            clipboardManager.setPrimaryClip(ClipData.newPlainText(null, s));
            Toast.makeText(this, "成功复制到剪贴板", Toast.LENGTH_SHORT).show();
            break;
        case R.id.paste:
            editText.setText(clipboardManager.getPrimaryClip().getItemAt(0).getText());
            break;
    }
}

这里定义了两个按钮。

用户点击复制按钮后，通过获取 CLIPBOARD_SERVICE 系统服务转化成 ClipboardManager 对象，再调用 setPrimaryClip() 方法给剪贴板赋值，该方法接收一个 ClipData 对象，通过 newPlainText(label, text) 生成，数据是文字 text，描述是 label。最后再给出相应的提示。

用户点击粘贴按钮后，通过获取 CLIPBOARD_SERVICE 系统服务转化成 ClipboardManager 对象，再从 ClipboardManager 中获取对应的内容粘贴到编辑框内。
